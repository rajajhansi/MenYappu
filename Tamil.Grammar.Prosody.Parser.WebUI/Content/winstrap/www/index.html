Changelog

0.5.7
  • Updated glyph fonts. Glyph style is now aligned with Windows 10.
  • Added new glyphs.
  • Removed fixed width from .container layout.
  • Prebuilt and minified CSS files. NPM and Bower packages now come with prebuilt dist folder and minified CSS files.
0.5.6
  • Simplified Jumbotron.
  • Added .compressed option to Navbar.
0.5.5
  • Big update to Navbar allowing two navbars to stack.
  • Added variable font path with !default declaration to enable you to set the path based on your project needs
0.5.3
  • Added side-nav control.
  • Moved bower to devDependency in package.json.
0.4.12
  • Updated toggle switch, tooltip, popover, and link button.
  • Added color themes to <a> and fixed states.
  • Changed color theme classes.
  • Updated breadcrumb styling.
  • Changed font-family variable names.
  • Updated glyphs font path.
  • Fixed checkbox glyph issue.
0.4.9
  • Typography:
  • Updated type ramp, classes, mixins, and functions.
  • Added type controls (headings, subheadings, paragraphs, and captions).

  • Glyphs:
  • Changed glyph names to be more semantic.
  • Added glyph function to get a glyph's unicode value.
  • Other changes:

  • Added Navbar component
  • Updated Dropdown control
